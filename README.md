
# 🚀 VirtunoRunner — Run Scripts from Your Browser (Python • Shell • Audio)

Run Python and shell scripts remotely from a clean web interface, with real-time output, file preview and integrated audio rendering.

Built with NiceGUI. Designed for developers, automation workflows and creative pipelines.

---

## 🧠 Why VirtunoRunner?

VirtunoRunner transforms your machine into a **remote execution engine**, accessible from any browser.

No SSH friction. No terminal juggling.  
Just open your browser — and run.

### 🔥 Key advantages

- 🌐 Execute scripts remotely from any device
- ⚡ Real-time output streaming
- 🎵 Integrated audio pipeline (MIDI → render → play)
- 🧩 Extend with your own scripts and workflows
- 🧑‍💻 Clean and simple UI (NiceGUI)
- 🔐 Multi-user ready (expandable)

---
